Подготовка:
1.Должна быть установлена IDEA
2.Должен быть установлен SDK 20

Запуск:
1. Подключить SDK 20 в File -> Project Structur -> Project  и  File -> Project Structur -> Modules
2. Подключить зависимости в  File -> Project Structur -> Modules (все зависимости находятся в папке "Zavisimosti", по идее там надо будет поставить галочку под пакетом <Module sours>, сохранить изменения)
3.Запускаем программу и ждём 24 часа
